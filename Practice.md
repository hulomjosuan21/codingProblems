## Reverse Index Iteration
```python
n = 5

i, j = 0, n-1

while i < n:
    print(f"{i} : {j}")

    i += 1
    j -= 1

# Output 
# 0 : 4
# 1 : 3
# 2 : 2
# 3 : 1
# 4 : 0
```
